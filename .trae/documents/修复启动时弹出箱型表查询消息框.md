# 修复计划：启动时弹出箱型表查询消息框

## 问题描述
程序启动时（进入主窗体前）会弹出一个消息框，显示"查询结果已保存到: xxx\箱型表查询结果.txt"。
这不是期望的行为，用户希望直接进入主窗体而不显示此消息框。

## 问题原因
在 [Program.cs](file:///h:/TraeDev/易捷查询CSharp/易捷查询CSharp/易捷查询CSharp/Program.cs#L19) 的 `Main` 方法中，第19行调用了：
```csharp
模块_通用函数.查询箱型表();
```

这个方法会：
1. 连接Oracle数据库
2. 查询所有表名和箱型相关表结构
3. 将结果保存到文本文件
4. **弹出 MessageBox 显示保存路径**（第242行）

## 解决方案

### 方案1：删除启动时的调用（推荐）
直接删除或注释掉 [Program.cs](file:///h:/TraeDev/易捷查询CSharp/易捷查询CSharp/易捷查询CSharp/Program.cs#L19) 第19行的调用。

**优点：**
- 简单直接
- 不影响其他功能
- 如果需要该功能，可以改为在菜单或按钮中调用

**缺点：**
- 如果这是调试用的代码，删除后无法自动执行

### 方案2：添加参数控制是否显示消息框
修改 `查询箱型表` 方法，添加一个参数控制是否弹出消息框。

**优点：**
- 保留功能，可灵活控制
- 不影响现有代码

**缺点：**
- 需要修改方法签名
- 稍微复杂一些

## 实施步骤

采用**方案1**，步骤如下：

1. 打开 [Program.cs](file:///h:/TraeDev/易捷查询CSharp/易捷查询CSharp/易捷查询CSharp/Program.cs)
2. 删除或注释掉第19行：`模块_通用函数.查询箱型表();`
3. 保存文件
4. 重新编译运行项目

## 预期结果
- 程序启动时直接进入主窗体（窗体_功能列表）
- 不再弹出"箱型表查询结果"消息框
- 其他功能正常运行
