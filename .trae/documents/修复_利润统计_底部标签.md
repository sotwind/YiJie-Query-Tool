# 修复计划：利润统计底部标签更新

## 问题发现

窗体底部有独立的标签控件需要更新：
- `标签_总单数`
- `标签_总报价`
- `标签_总卖价`
- `标签_总利润`
- `标签_平均利率`

这些标签在 `显示结果` 方法中没有被更新，所以一直显示默认值 "0"。

## 修复方案

在 `显示结果` 方法中，除了添加 ListView 的汇总行外，还需要更新底部的标签控件。

## 修改内容

在 `显示结果` 方法的最后，添加：
```csharp
// 更新底部汇总标签
标签_总单数.Text = 显示行数.ToString();
标签_总报价.Text = 总报价.ToString("0.00");
标签_总卖价.Text = 总卖价.ToString("0.00");
标签_总利润.Text = 总毛利.ToString("0.00");
标签_平均利率.Text = 平均利率.ToString("0.00") + "%";
```
