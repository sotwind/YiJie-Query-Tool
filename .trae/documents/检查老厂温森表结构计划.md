# 检查老厂和温森表结构计划

## 目标

检查老厂和温森的数据库表结构，找出：
1. ORD_BAS 表是否有其他日期字段（如 CRTDAT、CREATED 等）
2. 正确的表名和字段名
3. 为每个厂制定正确的基础表查询方案

## 检查步骤

### 1. 老厂数据库检查

使用 mcp_yijie-laochang 检查：
```sql
-- 查看 ORD_BAS 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'ORD_BAS' ORDER BY column_id;

-- 查看 ORD_CT 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'ORD_CT' ORDER BY column_id;

-- 查看 PB_CLNT 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'PB_CLNT' ORDER BY column_id;
```

### 2. 温森数据库检查

使用 mcp_yijie-wensen 检查：
```sql
-- 查看 ORD_BAS 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'ORD_BAS' ORDER BY column_id;

-- 查看 ORD_CT 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'ORD_CT' ORDER BY column_id;

-- 查看 PB_CLNT 表结构
SELECT column_name FROM user_tab_columns WHERE table_name = 'PB_CLNT' ORDER BY column_id;
```

### 3. 对比三个厂的表结构

找出差异：
- 日期字段名（PTDATE vs CRTDAT vs CREATED）
- 字段是否存在
- 表名是否一致

### 4. 制定查询方案

根据检查结果，为每个厂制定正确的基础表查询方案。

## 实施步骤

1. 使用老厂 MCP 检查表结构
2. 使用温森 MCP 检查表结构
3. 对比三个厂的差异
4. 修改代码，为每个厂使用正确的字段名
5. 测试验证
