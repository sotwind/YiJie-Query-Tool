# 排查计划：利润统计数据为0

## 问题分析

用户反馈汇总行显示为0，而且 `dataList.Sum` 也是0。

这说明问题不在汇总逻辑，而是：
1. SQL 查询没有返回数据
2. 或者 SQL 查询返回的数据中金额为0

## 排查步骤

1. 检查 SQL 是否能正确执行
2. 检查 SQL 返回的数据是否有值
3. 检查字段映射是否正确

## 可能的原因

1. SQL 中的字段名大小写问题（Oracle 默认大写）
2. 关联条件不正确导致没有匹配数据
3. 筛选条件过于严格导致没有数据
4. 日期格式问题

## 需要验证的 SQL

新系统：
```sql
SELECT 
    NVL(b.QUOPRC, 0) * NVL(v.ACCNUM, 0) as 报价总金额
FROM V_ORD v
LEFT JOIN ORD_BAS b ON v.SERIAL = b.SERIAL AND v.CLIENTID = b.CLIENTID AND v.ORGCDE = b.ORGCDE
WHERE v.ISACTIVE = 'Y'
  AND v.PTDATE >= to_date('2026-03-06', 'yyyy-MM-dd')
  AND v.PTDATE < to_date('2026-03-07', 'yyyy-MM-dd')
```

需要验证：
1. V_ORD 和 ORD_BAS 的关联条件是否正确
2. QUOPRC 和 ACCNUM 是否有值
3. 是否有数据满足条件
