## 技术要点

### 数据库连接
- 使用 `DatabaseInfos.GetDatabaseInfos()` 获取所有数据库连接
- 遍历各服务器查询数据
- 使用 `SqlHelperFactory.OpenDatabase` 连接 Oracle 数据库

### 表结构参考
- **报价表**: MkPrctyp 系列（02_报价模块.md）
- **订单表**: OrdCt 系列（03_订单模块.md）
- **业务员表**: pb_dept_member（新系统）
- **客户表**: PbClnt 系列（01_客户管理模块.md）
- **产品表**: Prd 系列（05_产品物料模块.md）

### 关键 SQL 示例
```sql
SELECT 
    t.ptdate as 日期，
    t.serial as 单号，
    c.clntnme as 客户，
    p.prdtnme as 产品，
    e.empnme as 业务员，
    d.dptnme as 部门，
    nvl(sum(t.accamt),0) as 报价金额，
    nvl(sum(s.sell_amt),0) as 卖价金额，
    nvl(sum(s.sell_amt),0) - nvl(sum(t.accamt),0) as 利润差额，
    case when nvl(sum(t.accamt),0) = 0 then 0
         else (nvl(sum(s.sell_amt),0) - nvl(sum(t.accamt),0)) / nvl(sum(t.accamt),0) * 100
    end as 利率
FROM v_ord t
LEFT JOIN pb_clnt c ON t.clntcde = c.clntcde
LEFT JOIN prd_base p ON t.prdtcde = p.prdtcde
LEFT JOIN pb_dept_member e ON t.agntcde = e.mobile
LEFT JOIN pb_dept d ON e.dptcde = d.dptcde
LEFT JOIN sell_detail s ON t.serial = s.order_no
WHERE t.status = 'Y'
  AND t.ptdate >= to_date(:start_date, 'yyyy-MM-dd')
  AND t.ptdate < to_date(:end_date, 'yyyy-MM-dd')
GROUP BY t.ptdate, t.serial, c.clntnme, p.prdtnme, e.empnme, d.dptnme
```
