## 需求分析

### 2.2 排除未审核数据（已报价但未生成工单）

**条件**：

* 报价金额 > 0（已报价）

* 卖价金额 = 0（未填售价）

* 这些记录需要剔除

**SQL 实现**：

```sql
-- 排除未审核数据（已报价但未填售价）
AND NOT (NVL(b.QUOPRC, 0) > 0 AND NVL(b.PRICES, 0) = 0)
-- 或者只保留有卖价的记录
AND NVL(b.PRICES, 0) > 0
```

### 2.3 排除特定数据（Z开头单号 + 李仰忠）

**条件**：

* 单号以 'Z' 开头

* 业务员 = '李仰忠'

**SQL 实现**：

```sql
-- 排除 Z 开头且业务员为李仰忠的数据
AND NOT (b.SERIAL LIKE 'Z%' AND c.AGNTCDE = '李仰忠的业务员编码')
```

### 2.4 客户名称和业务员转换

**条件**：

* 客户名称 = '森林包装集团股份有限公司'

* 业务员 = '李仰忠'

**转换**：业务员显示为 '莫梦辉'

**SQL 实现**：

```sql
-- 使用 CASE 转换业务员
CASE 
    WHEN c.CLNTNME = '森林包装集团股份有限公司' AND c.AGNTCDE = '李仰忠的业务员编码' 
    THEN '莫梦辉' 
    ELSE c.AGNTCDE 
END as 业务员编码
```
