# 移除订单号包含 "." 的数据计划

## 需求

在利润统计页面，移除订单号（SERIAL）中包含 "." 的数据。

## 实现方案

在所有厂的 SQL 查询中，添加排除条件：`AND SERIAL NOT LIKE '%.%'`

## 修改位置

文件：`h:\TraeDev\易捷业务数据查询\易捷查询CSharp\易捷查询CSharp\窗体_利润统计.cs`

需要修改三个分支的 WHERE 条件：

### 1. 新厂（新厂新系统）

```sql
WHERE t.ISACTIVE = 'Y'
  AND b.STATUS = 'Y'
  AND NOT (b.SERIAL LIKE 'Z%' AND c.AGNTCDE = '13655860812')
  AND b.SERIAL NOT LIKE '%.%'  -- 排除包含 "." 的订单号
  AND b.PTDATE >= ...
```

### 2. 老厂和温森（新系统）

```sql
WHERE v.STATUS = 'Y'
  AND NOT (v.SERIAL LIKE 'Z%' AND v.AGNTCDE = '13655860812')
  AND v.SERIAL NOT LIKE '%.%'  -- 排除包含 "." 的订单号
  AND v.PTDATE >= ...
```

### 3. 临海（旧系统）

```sql
WHERE v.STATUS = 'Y'
  AND NOT (v.SERIAL LIKE 'Z%' AND v.AGNTCDE = '13655860812')
  AND v.SERIAL NOT LIKE '%.%'  -- 排除包含 "." 的订单号
  AND v.PTDATE >= ...
```

## 实施步骤

1. 修改新厂分支的 SQL，添加 `AND b.SERIAL NOT LIKE '%.%'`
2. 修改老厂和温森分支的 SQL，添加 `AND v.SERIAL NOT LIKE '%.%'`
3. 修改临海分支的 SQL，添加 `AND v.SERIAL NOT LIKE '%.%'`
4. 编译测试

## 注意事项

* 使用 `NOT LIKE '%.%'` 可以排除任何包含 "." 的字符串

* 确保在所有三个分支中都添加此条件
