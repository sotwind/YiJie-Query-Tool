3. **数据库关联关系**
   ```
   ord_bas → pb_clnt → pb_clnt_agnt → pb_dept_member
     │          │           │              │
   clntcde   clntcde    clntcde        empcde
                                         │
                                       dptnme (部门名称)
   ```

4. **核心查询 SQL**
   ```sql
   SELECT 
       b.serial, b.created, b.clntcde, c.clntnme,
       ca.agntcde, dm.empnme, dm.dptnme,
       b.quoprc, b.prices, b.ordnum, b.accamt,
       b.accamt - (b.quoprc * b.ordnum) as diff,
       差额率计算
   FROM ferp.ord_bas b
   LEFT JOIN ferp.pb_clnt c ON b.clntcde = c.clntcde
   LEFT JOIN ferp.pb_clnt_agnt ca ON c.clntcde = ca.clntcde AND ca.isactive = 'Y'
   LEFT JOIN ferp.pb_dept_member dm ON ca.agntcde = dm.empcde AND dm.isactive = 'Y'
   WHERE b.isactive = 'Y' AND b.quoprc IS NOT NULL AND b.quoprc > 0
   ```

## 📊 功能说明

### 筛选条件
- **服务器**：选择查询的数据库（新厂新系统/老厂新系统/温森新系统）
- **日期范围**：按订单创建日期筛选
- **业务员**：下拉选择（支持全部）
- **部门**：下拉选择（支持全部）
- **客户**：下拉选择（支持全部）
- **单号**：模糊搜索
- **产品**：按产品代码或名称模糊搜索

### 统计结果
| 列名 | 说明 |
|-----|------|
| 服务器 | 数据来源数据库 |
| 单号 | 生产单号 |
| 客户 | 客户名称 |
| 业务员 | 业务员姓名 |
| 部门 | 所属部门 |
| 产品代码 | 产品编码 |
| 产品名称 | 产品名称 |
| 订单数量 | 订单数量 |
| 报价单价 | 报价系统单价 (quoprc) |
| 实际单价 | 实际成交单价 (prices) |
| 报价金额 | 报价单价 × 订单数量 |
| 卖价金额 | 订单总金额 (accamt) |
| **差额** | 卖价金额 - 报价金额（红色=亏损，绿色=盈利） |
| **差额率** | 差额 / 报价金额 × 100% |

### 底部统计
- 记录总数
- 报价金额合计
- 卖价金额合计
- 差额合计

### 导出功能
- 点击"导出 Excel"按钮可将结果导出到 Excel

## ⚠️ 注意事项

1. **临海系统**：暂未支持（旧系统数据库结构不同）
2. **数据权限**：使用 read 用户，只能查询不能修改
3. **性能优化**：大量数据时查询可能需要几秒
4. **客户列表**：由于客户数量较多（5000+），目前未预加载，后续可支持搜索
