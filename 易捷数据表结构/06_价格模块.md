# 价格模块表结构

本模块包含价格相关的所有表结构信息，涵盖销售费用、毛利率、税费、报价公式、运费等。

---

## 1. MkSelling 销售费用表

**表名**: mk_selling

**说明**: 销售费用表

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String | 更新人 |
| orgcde | String | 组织代码 |
| clientid | String | 客户ID |
| isactive | String | 是否启用 |
| clntyp | String | 客户类型-FIX115--ST 三级厂(报价销售费特殊处理降点),OT=其它,默认CS=终端（销售费特殊处理） |
| period | Integer | period付款期（公共定义，不使用系统定义）30天月结之类DYU020 |
| bsrate | BigDecimal | 销售费率 |
| remark | String | 备注说明 |

## 5. PbTypeExp 客户算料公式表

**表名**: pb_type_exp

**说明**: 客户算料公式表

| 字段名 | 类型 | 中文注释 |
