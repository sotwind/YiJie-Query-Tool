# 工艺模块表结构

本模块包含工艺相关的所有表结构信息，涵盖工厂生产工序、报价工序、产品工序、盒型设计等。

---

## 1. PbProcesses 工厂生产工序

**表名**: pb_processes

**说明**: 工厂生产工序

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String(11) | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String(11) | 更新人 |
| orgcde | String(5) | 组织代码 |
| clientid | String(5) | 客户ID |
| isactive | String(1) | 是否有效 |
| prccde | String(10) | 工序代码 |
| prcnme | String(40) | 名称 |
| prctyp | String(2) | 工序类型(CTC001) |
| isplan | String(1) | 是否进行排程 Y/N |
| isouts | String(1) | 是否外协 |
| ismkts | String(1) | 销售建档是否展示 |
| frsize | String(1) | 外协使用的规格 |
| quence | Integer | 排序 |
| maxl | Integer | 最大长度 |
| minl | Integer | 最小长度 |
| maxw | Integer | 最大宽度 |
| minw | Integer | 最小宽度 |
| sminw | Integer | 每切最小宽度 |
| maxprs | Integer | 最大压线条数 |
| splitw | Integer | 最大开数 |
| bleeds | Integer | 最小修边;出血尺寸 |
| debugs | Integer | 调试时间（m） |
| capacity | Integer | 每小时产能 |
| speeds | Integer | 速度 |
| teches | String(200) | 要求 |
| npbleedw | Integer | 不压线修边 |
| prcost | BigDecimal | 加工费 |
| prcutl | String(2) | 加工费单位PC=个价，M2=平方价 |
| remark | String(80) | 备注 |
| maccde | String(15) | 机器代码 |
