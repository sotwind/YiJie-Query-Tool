## 模块说明

### 01 客户管理模块 (PbClnt系列)
包含客户基本资料、地址、联系人、财务信息、信用额度、材质报价等13张表。

### 02 报价模块 (MkPrctyp/MsPrctyp系列)
- **MkPrctyp系列**: 25张报价表，涵盖纸板、水印、粘箱、胶印、覆膜、模切、钉箱等工艺
- **MsPrctyp系列**: 18张放损表，定义各工序的损耗计算规则

### 03 订单模块 (OrdCt系列)
包含订单主表、订单明细、订单状态、订单数量、订单日期、订单用料、订单工序等9张表。

### 04 工艺模块
包含工厂生产工序、报价工序、产品工序、盒型设计、APS排程等5张表。

### 05 产品物料模块
包含产品基础信息、产品BOM、产品设计、印版模板、瓦楞坑型等12张表。

### 06 价格模块
包含销售费用、毛利率、税费、报价公式、运费等10张表。

### 07 机器模块
包含设备定义、设备纸度、设备养护、设备联结等4张表。

### 08 系统管理模块
包含用户管理、角色管理、菜单管理、组织管理、日志管理等11张表。

### 09 其他模块
包含报价单、纸板报价、班组、盒型、车辆、文件、质检等19张表。
