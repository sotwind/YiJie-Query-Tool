# 报价模块表结构

本模块包含报价相关的所有表结构信息，分为两类：
- **MkPrctyp系列**：报价表，用于存储各个工序的报价信息
- **MsPrctyp系列**：放损表，用于存储各个工序的放损（损耗）信息

---

# MkPrctyp 系列表（报价表）

---

## 1. MkPrctyp01 纸板报价表

**表名**: mk_prctyp01

**说明**: 纸板报价表（或是彩印组合报价表）

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String | 更新人 |
| orgcde | String | 组织代码 |
| clientid | String | 客户ID |
| isactive | String | 是否有效 |
| clntyp | String | 客户类型（=客户表中的类型） |
| prcmod | String | 计价方法（FIX156/LP=加工费/PI=计件/WX=外协) |
| status | String | 是否有效 |
| matcde | String | 纸质 |
| invcde | String | 客户材质 |
| crrcde | String | 楞别(注意，该处的坑型必须是5BC = 层数 +坑型的形式) |
| layers | String | 层数 |
| cstprc | BigDecimal | 成本价格 |
| fixfee | BigDecimal | 起做费，开机费 |
| prices | BigDecimal | 真正的纸板报价 |
| remark | String | 名称 |
| minamt | BigDecimal | 保底 |

## 2. MkPrctyp02 水印报价

**表名**: mk_prctyp02

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String | 更新人 |
| orgcde | String | 组织代码 |
| clientid | String | 客户ID |
| isactive | String | 是否有效 |
| clntyp | String | 客户类型 |
| clntcde | String | 客户编码C0001（A开头） |
| prcmod | String | 计价方法（FIX156/LP=加工费/PI=计件/WX=外协,CO=成本计算) |
| prccde | String | 工序代码 |
| prcnme | String | 名称 |
| mactyp | String | 机型（联动线=LDX，淘宝=BAO） |
| prtacr | String | 印刷面积（30D,50D,70D,70U） |
| tptype | String | 模板类型Y=圆模 P=平模=,tptype = 模板类型 ，YM=圆模，PM=平模，KC=开槽，NO=无 |
| oiltyp | String | 过油种类（红外，UV，防剥离，吸塑，防潮，亚光，局部UV） |
| colsum | Integer | 列印颜色 color数量=1-9（线数） |
| caltyp | String | 计算方式（PCS=个,ACR=面积） |
| minnum | Integer | 最低数量 |
| maxnum | Integer | 最高数量 |
| minacr | BigDecimal | 最低面积 |
| maxacr | BigDecimal | 最高面积 |
| sumacr | BigDecimal | 开槽计价需要按总面积（低于1000，高于1000） |
| fixfee | BigDecimal | 起做费，开机费 |
| minamt | BigDecimal | 保底 |
| fixacr | BigDecimal | 最低面积（如果产品面积低于该面积，则以最低面积计算） |
| prices | BigDecimal | 价格（注意单位） |
| spcamt | BigDecimal | 专色附加费（按每次计算） |
| motamt | BigDecimal | 杂色附加费（按每次计算） |
| remark | String | 备注说明 |
| layers | String | 层数（3、5、7） |
