# 机器模块表结构

本模块包含机器设备相关的所有表结构信息，涵盖设备定义、设备纸度、设备养护、设备生管联结等。

---

## 1. PbMachine 设备定义表

**表名**: pb_machine

**说明**: 设备定义表

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String | 更新人 |
| orgcde | String | 组织代码 |
| clientid | String | 客户ID |
| isactive | String | 是否启用 |
| maccde | String | 机器代码 |
| macnme | String | 名称 |
| mactyp | String | 机型(或是尺寸规格)，一般用于报价描述机型 |
| ip | String | IP地址 |
| crrcde | String | 楞别组合 5BA/3B/3E/7BAB/7EBC |
| ptdate | Date | 装备或是购进日期 |
| accamt | BigDecimal | 购置金额 |
| specs | String | 规格型号 |
| prvnme | String | 供应商 |
| maxl | Integer | 最大长度 |
| minl | Integer | 最小长度 |
| maxw | Integer | 最大宽度 |
| minw | Integer | 最小宽度 |
| sminw | Integer | 每切最小宽度 |
| splits | Integer | 最大开数 刀数 - 1 |
| maxprs | Integer | 最大压线条数 |
| bleeds | Integer | 最小修边;出血尺寸，二边的合计值 |
| npbleedw | Integer | 不压线修边 |
| quence | Integer | 排序 |
| lssrat | BigDecimal | 许可损失比例 |
| lssmax | BigDecimal | 许可损失比例上限，生产数量超出此处，不再按比例计算 |
| lssfix | BigDecimal | 许可损失比例上限，生产数量超出此处，不再按比例计算 |
| prcost | BigDecimal | 加工费 |
| tmpsts | String | 是否可以模切 |
| prccde | String | 工序代码 |
| prctyp | String | CTC001(工序类型) |
| wkscde | String | 车间编码 Workshop |
| debugs | Integer | 调试时间（m） |
| capacity | Integer | 最多能排产时间（分钟数） |
| colnum | Integer | 列印颜色 color数量=1-9 |
| autmac | String | 是否是带生管设备 = Y |
| revsts | String | 生管数据是否主动回收 Y |
| objtyp | String | 类别 |
| speeds | Integer | 速度/分 |
| places | String | 放置地点 |
| person | String | 维护管理人 |
| remark | String | 备注 |
| buydat | Date | 购置日期 |
| startup | String | 开机时间 starting up... |
| macico | String | 图标 |
| weburl | String | webservice地址 |
| facara | String | 生产厂区Factory area=CTC170车间 |
| params | Integer | 生产片数,一页一片,二页二片,四页四片 |

## 4. PbMachineUrl 设备生管联结信息表

**表名**: pb_machine_url

**说明**: 设备生管联结信息表

| 字段名 | 类型 | 中文注释 |
|--------|------|----------|
| id | Integer | 主键 |
| created | Date | 创建时间 |
| createdby | String | 创建人 |
| updated | Date | 更新时间 |
| updatedby | String | 更新人 |
| orgcde | String | 组织代码 |
| clientid | String | 客户ID |
| isactive | String | 是否启用 |
| mid | Integer | 防止修改设备代码 |
| mesurl | String | 生管系统本身的服务地址(或是生产中心地址) |
| maccde | String | 机器代码 |
| driver | String | JDBC驱动 |
| url | String | 数据库连接地址 |
| dyip | String | dynamic ip |
| usrcde | String | 用户名 |
| passwd | String | 密码 |
| upmode | String | 上传模式CTC080/TXT/DB |
| tmpsto | String | temporary storage;许可临时存放服务器，开机后自取 |
| upfile | String | 上传文件名 |
| updir | String | 上传目录名称 |
| dwmode | String | 下载模式 CTC080/TXT/DB |
| dwfile | String | 下载文件名或是表名 |
| dwtble | String | 下载真实表名（真正的更新表名） |
| separator | String | 分隔符 |
| matsep | String | 材质分隔符或长度(如果为数字，则为长度，比如京山的材质为二位长度，而有些生管则需要/分隔) |
| dwdir | String | 下载的目录名称 |
| dwskey | String | 回收后更新主键字段名 |
| charset | String | 生成文本文件的字符集（默认为空就好） |
| dwcharset | String | 回收文件的字符集（为空取charset） |
| dwkfld | String | 回收表（生管表）主键字段名（第一行一定为主键字段） |
| dwdata | String | 回收后更新标识类型Y/true之类 |
| remark | String | 备注 |
| cancel | String | 取消方法（E/I）(E=易捷)(I=插入) |
| seekdate | String | 读取值日期 |
| seek | Integer | 读取值 |
| macurl | String | 设备前端地址(JSON接口地址) |
| txmode | String | 文本模式，FTP，SMB |
| ftport | Integer | FTP端口 |
| isexch | String | 是否将待生产数据同步到ERP供排车使用 |
| issign | String | JSON模式排产时是否是批量发送订单 |
